<details>
  <summary>How To Make A Class</summary>
  
  <br>
  This Is A Basic Class<br>
  
  
  ```python
  # Make A Class
  class MyClass:
  
    # Make A Function
    def say_hello(self, name):
      print(f"Hello {name}")
  ```
  
</details>

<details>
  <summary>How To Use A Class</summary>
  
  <br>
  This Is How You Use A Object In Python<br>
  
  
  ```python
  
  class MyClass:
  
    def say_hello(self, name):
      print(f"Hello {name}")
  
  # Initilize Object
  function = MyClass()
  
  # Use Object
  function.say_hello("GitHub")
  ```
  
</details>
